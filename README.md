# LightDrop

LightDrop — маленький нежный JPG-компрессор, который помогает быстро подготовить изображения для публикации.

Почему появился этот проект?

Потому что Ася не любит Photoshop.

Каждый раз открывать огромную программу только ради «Сохранить для Web» — долго и неудобно. Поэтому появился LightDrop: загружаешь изображеньице, нажимаешь «Чпоньк» и получаешь лёгкий JPG.

## Что умеет

- аккуратно уменьшает изображение;
- ограничивает размер до 2000 px по меньшей стороне;
- сохраняет результат в JPG;
- работает прямо в браузере;
- не требует настроек;
- делает всё через одну кнопку «Чпоньк».

## Маленький секрет

Если нажать на сердечко, можно получить короткое романтичное признание.

Эта часть работает через Cloudflare Pages Function и Cerebras API. API-ключ хранится только в Cloudflare Secret.
